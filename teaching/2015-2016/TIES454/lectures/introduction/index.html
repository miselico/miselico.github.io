Main targets of the course

  • Learn what an agent-based system is
    • By using basic components
  • Learn how agents can plan and collaborate
    • By using BDI architectures and auctions
  • Learn what an agent-based system can be used for
    • By implementing scenarios
  • Learn how agent-based systems connect to other technologies
    • By implementing external behavior for an agent

Side targets

  • Learn concurrent/parallel programming
    • Bill Dally (Chief scientist of NVidia) :

    The future of the world is parallel. You know, Serial programs it’s going to be like COBOL programs. People used to do those, but they don’t do them anymore.

  • Learn the Git distributed version control system
  • Learn how agent-based systems relate to other technologies
  • Use English as a communication language
  • Perform group work with new people

  • Enjoy!

Why do you want to study Multi-Agent Systems?

  • Rarely for the systems themselves
  • Mainly to carry over concepts to other technologies
    • Messaging systems
    • Concurrency models
    • General AI
    • Machine learning
    • SOA
    • NLP
    • …
  • To know what the future might hold
